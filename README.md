\# API Performance Monitor



A Flask-based web application to monitor API response times, status codes, and errors.



\## Features

\- Check API endpoint status

\- Measure response time

\- Store logs in SQLite

\- Show total checks, success count, and average response time



\## Tech Stack

\- Python

\- Flask

\- Requests

\- SQLite



\## Run the project

```bash

pip install -r requirements.txt

python app.py

```
